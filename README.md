# 💀 BROOK — Soul King AI Companion

> *"YOHOHOHOHO! Your personal AI companion, Captain."*

A premium personal AI companion with a soul aesthetic. Built with HTML, CSS, and Vanilla JavaScript.

## ✨ Features

- 🤖 **3 AI Models** — Claude, Gemini, Groq (switch anytime)
- 💬 **Modern Chat** — Markdown, code blocks with copy button, typing indicator
- 🧠 **Memory System** — Notes, Ideas, Projects, Saved Chats
- 👻 **Soul Mode** — Ghost animation while thinking
- 🎵 **Music Notes** — Floating ambient notes
- 🔊 **Voice** — Web Speech API (optional)
- 💀 **Wanted Poster** — Premium profile page
- ✨ **Particle System** — Canvas-based ambient particles
- 📱 **Mobile First** — Fully responsive

## 🚀 Setup

1. Clone or download this repo
2. Open in browser (or deploy to GitHub Pages)
3. Tap ⚙️ and add your API key(s):
   - **Claude**: [console.anthropic.com](https://console.anthropic.com)
   - **Gemini**: [aistudio.google.com/apikey](https://aistudio.google.com/apikey) *(Free)*
   - **Groq**: [console.groq.com](https://console.groq.com) *(Free)*
4. Start chatting!
